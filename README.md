# TinyGPT – Mini Transformer from Scratch

## 🚀 Overview
A lightweight GPT-style language model built from scratch using PyTorch to understand Transformer architecture deeply.

## 🧠 Features
- Multi-head self-attention
- Positional encoding
- Decoder-only Transformer
- Custom training loop
- Text generation

## ⚙️ Tech Stack
Python • PyTorch

## ▶️ Run
python train.py  
python generate.py

## 💡 Goal
Understand how GPT-style models work internally.
